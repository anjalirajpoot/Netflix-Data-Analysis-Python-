# Netflix-Data-Analysis-Python-
This project focuses on analyzing the Netflix dataset using Python. The main objective was to understand content distribution, trends over time, and popular genres. The project demonstrates practical skills in data analysis, data visualization, and extracting meaningful insights from real-world datasets.
# Netflix Data Analysis 📊

## 📌 Project Overview

This project analyzes Netflix dataset using Python to find trends and insights.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Key Insights

* More Movies than TV Shows
* Content increased after 2015
* USA has highest content
* Drama is most popular genre

## 📁 Dataset

Downloaded from Kaggle

## 🚀 Conclusion

This project helped in understanding data cleaning, analysis, and visualization.
