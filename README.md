# Netflix Data Analysis 📊

## 📌 Project Overview

This project analyzes Netflix dataset using Python to find trends and insights.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Key Insights

* More Movies than TV Shows
* Content increased after 2015
* USA has highest content
* Drama is most popular genre

Downloaded from Kaggle
